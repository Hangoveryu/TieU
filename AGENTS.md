# AGENTS.md — 历史粘贴板项目开发指引

## 项目简介

这是一个 Windows 桌面应用——**历史粘贴板**。它自动记录用户剪贴板中的文字和图片，以卡片形式按时间倒序展示，支持搜索、置顶、删除和自动过期清理。

## 技术栈

- **框架**：Electron（桌面端跨平台框架）
- **数据库**：better-sqlite3（嵌入式 SQLite）
- **图片处理**：sharp（缩略图生成）
- **打包**：electron-builder（Windows 安装程序）
- **语言**：JavaScript（前后端统一）

## 工作规范

### 每次开发会话

1. **开始前** — 阅读 `docs/执行计划.md`，确认当前应该做哪一步
2. **编码前** — 阅读对应的规范文档（UI 看 design-spec，数据看 tech-spec）
3. **完成后** — 更新 `devlog/YYYY-MM-DD.md`，记录今日完成和待办事项
4. **如果发现规范有问题** — 先更新对应的 docs/ 文档，再修改代码

### 稳步推进原则

- **一次只做一个步骤**，每步完成后暂停，等待用户确认再继续
- **每步有明确验收标准**，通过验收才算完成
- **不在计划外添加功能**，新想法记录到需求文档的 P2 部分

### 代码风格

- **缩进**：2 空格
- **引号**：单引号 `'`
- **分号**：不强制，但同一文件内保持一致
- **命名**：文件 `kebab-case`，变量 `camelCase`，常量 `UPPER_SNAKE_CASE`
- **注释**：复杂逻辑加注释说明"为什么这样做"，而不是"做了什么"

### Git 提交

- 每完成一个子任务就提交一次
- 提交信息格式：`步骤N: 简短描述`（如 `步骤3: 实现数据库CRUD操作`）
