# 需求规格说明

> 版本：v1.0 | 更新日期：2026-06-08

## 项目概述

**历史粘贴板** 是一款 Windows 桌面应用，自动记录用户剪贴板中的文字和图片，以卡片形式按时间倒序展示历史记录，支持搜索、置顶、删除和自动过期清理。

## 用户角色

| 角色 | 描述 |
|------|------|
| 普通用户 | 需要频繁复制粘贴的电脑使用者，期望能快速回溯最近的复制历史 |

## 功能需求

### P0 — 核心功能（必须有）

| 编号 | 功能 | 描述 |
|------|------|------|
| F1 | 剪贴板监听 | 自动监控并记录用户复制的所有文字和图片内容 |
| F2 | 历史卡片列表 | 以卡片形式按时间降序排列所有复制内容 |
| F3 | 复制/粘贴历史内容 | 点击卡片可将该内容重新复制到剪贴板；若面板由输入框附近唤起，则点击卡片后自动粘贴到该输入框 |

### P1 — 重要功能（第一期实现）

| 编号 | 功能 | 描述 |
|------|------|------|
| F4 | 存储期限设置 | 用户可设定保留天数：1天 / 3天 / 5天，过期自动清理 |
| F5 | 卡片置顶 | 重要卡片可以置顶，置顶卡片显示在最前面且不会被自动清理 |
| F6 | 卡片删除 | 用户可手动删除不需要的历史卡片 |
| F7 | 搜索功能 | 支持关键词实时搜索历史文字内容（不区分大小写） |
| F8 | 全局快捷键 | 按 `Ctrl+Shift+Z` 弹出/隐藏主面板；若当前焦点是输入框，则在输入框附近弹出或刷新目标输入框，否则在屏幕右下角弹出 |
| F9 | 开机自启 | 电脑开机后自动在后台运行 |
| F9.1 | 面板固定与隐藏 | 勾选固定后面板保持置顶，失焦和点击卡片自动粘贴后不自动隐藏；右上角 `×` 只隐藏面板，不退出程序 |

### P2 — 后续考虑（第二期+）

| 编号 | 功能 | 描述 |
|------|------|------|
| F10 | 快捷键自定义 | 用户可自定义唤起快捷键 |
| F11 | 多选批量删除 | 同时选中并删除多条记录 |
| F12 | 导出记录 | 将历史记录导出为文本文件 |
| F13 | 暗色模式 | 支持深色主题切换 |

## 非功能性需求

| 编号 | 类别 | 要求 |
|------|------|------|
| N1 | 性能 | 历史记录 1000 条以内列表滚动流畅（60fps） |
| N2 | 响应速度 | 复制后 1 秒内卡片出现在列表中 |
| N3 | 资源占用 | 后台运行时内存占用 < 100MB |
| N4 | 存储 | 图片保留原图质量，不压缩 |
| N5 | 稳定性 | 崩溃后下次启动能自动恢复数据 |
| N6 | 兼容性 | 支持 Windows 10 / Windows 11 |

## UI 设计要求

| 编号 | 要求 |
|------|------|
| U1 | 主色调：淡蓝色（约 `#B3D9FF` ~ `#5B9BD5` 范围） |
| U2 | 风格：简洁、直观，避免复杂操作层级 |
| U3 | 卡片布局：圆角矩形，信息层次清晰 |
| U4 | 操作入口：鼠标悬停显示操作按钮，日常隐藏保持简洁 |

## 约束条件

- 不加密存储数据（用户明确要求）
- 数据完全存储在本地，不上传网络
- 开发语言：JavaScript（Electron 框架）
