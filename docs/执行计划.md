## 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| native 模块编译失败 | 阻塞第 2 步 | 使用 electron-rebuild，锁定 Electron 与 native 模块兼容版本 |
| 剪贴板轮询性能问题 | 第 4 步 | 使用内容哈希快速去重，控制轮询频率 |
| 大量图片占用磁盘 | 第 3/4 步 | 配合保留天数自动清理旧图 |
| 全局快捷键冲突 | 第 5 步 | 检测冲突并提示用户，支持自定义快捷键 |
| Windows 版本兼容 | 第 7 步 | 在 Win10/Win11 上分别测试 |
