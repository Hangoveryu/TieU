# UI 设计规范

> 版本：v1.0 | 更新日期：2026-06-08

## 设计原则

- **简洁优先**：减少视觉噪音，只展示必要信息
- **操作直觉**：常用操作一键可达，不嵌套深层菜单
- **视觉层次**：通过颜色、间距、大小区分信息优先级

## 色彩规范

### 主色调：淡蓝色系

| 用途 | 色值 | 色块 | 说明 |
|------|------|------|------|
| 主背景 | `#F0F6FC` | ████ | 窗口整体背景，极淡蓝白 |
| 卡片背景 | `#FFFFFF` | ████ | 白色卡片，干净清晰 |

| 主色调 | `#5B9BD5` | ████ | 按钮、选中态、强调元素 |
| 主色悬停 | `#4A8AC4` | ████ | 鼠标悬停时加深 |
| 文字主色 | `#333333` | ████ | 正文文字，保证可读性 |
| 文字辅助 | `#888888` | ████ | 时间戳、次要信息 |
| 置顶标记 | `#FFB74D` | ████ | 金色星星，温暖醒目 |
| 删除按钮 | `#E57373` | ████ | 红色，警示操作 |
| 搜索边框 | `#B3D4F0` | ████ | 搜索框边框，略深于卡片边框 |

| 状态 | 色值 | 用途 |
|------|------|------|

## 交互规范

### 卡片操作
| 操作 | 触发方式 | 反馈 |
|------|----------|------|
| 复制/粘贴内容 | 点击卡片任意位置 | 若存在快捷键唤起时检测到的输入框，则自动粘贴并提示"已粘贴"；否则复制到剪贴板并提示"已复制" |
| 查看大图 | 点击图片卡片缩略图 | 弹出大图预览浮层 |
| 置顶/取消 | 点击 ☆ 按钮 | 星标填满/空心，卡片进入/离开顶部"置顶记录"分组；置顶记录不进入"更早的记录"分组 |
| 删除 | 点击 🗑 按钮 | 卡片滑出消失，出现"已删除"提示 |
| 操作按钮 | 鼠标悬停卡片时显示 | 按钮淡入出现 |

### 面板操作
| 操作 | 触发方式 | 反馈 |
|------|----------|------|
| 固定面板 | 勾选标题栏"固定" | 窗口保持置顶，失焦不隐藏，点击卡片自动粘贴后也不隐藏，并可连续粘贴到同一输入框 |
| 手动隐藏 | 点击标题栏右上角 × | 隐藏面板，不退出程序 |

### 搜索
| 操作 | 触发方式 | 反馈 |
|------|----------|------|
| 搜索 | 输入关键词 | 实时过滤，不区分大小写 |
| 清空搜索 | 点击搜索框右侧 × 按钮 | 恢复完整列表 |
| 无结果 | 搜索无匹配 | 显示"没有找到相关内容"空状态 |

### 快捷键
| 快捷键 | 作用 |
|--------|------|
| `Ctrl+Shift+Z` | 唤起 / 隐藏主面板；从外部输入框触发时优先显示在输入框附近并刷新自动粘贴目标，无输入框时显示在屏幕右下角 |
| `Esc` | 隐藏主面板（面板获得焦点时） |

### 动画
| 场景 | 动画 | 时长 |
|------|------|------|
| 面板出现 | 从鼠标附近位置淡入 + 轻微上移 | 200ms |
| 面板隐藏 | 淡出 | 150ms |
| 新卡片 | 从上方滑入 | 250ms |
| 删除卡片 | 向右滑出 + 淡出 | 200ms |
| 复制反馈 | 卡片边框闪烁蓝色 | 300ms |
