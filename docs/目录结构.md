| `CLAUDE.md` | AI 助手的总指引，包含规范路径和工作约定 | 项目规范变更时 |
| `AGENTS.md` | Codex/通用 AI 助手指引 | 项目规范变更时 |
| `README.md` | GitHub 项目说明和开发入口 | 对外说明变化时 |
| `LICENSE` | 开源许可证 | 许可证变更时 |

| `electron-builder.yml` | 打包为 Windows .exe 的配置 | 打包参数调整时 |
| `docs/需求规格.md` | 需求真相来源 | 需求变更时 |
| `docs/技术规范.md` | 技术决策与架构真相来源 | 技术方案调整时 |
| `docs/设计规范.md` | UI 设计真相来源 | 界面样式调整时 |

| `src/main/index.js` | 应用生命周期、窗口、托盘、快捷键 | 调整启动行为时 |
| `src/main/clipboard.js` | 剪贴板监听逻辑 | 修改监听策略时 |
| `src/main/store.js` | 数据库 CRUD | 修改数据模型时 |
| `src/main/autoclean.js` | 过期清理逻辑 | 调整清理规则时 |
| `src/main/autostart.js` | 开机自启注册 | 调整自启策略时 |

| `src/renderer/style.css` | 全局样式和主题 | 修改颜色/字体/间距时 |
| `src/renderer/app.js` | 前端主逻辑、IPC 通信 | 调整交互流程时 |
| `src/renderer/components/` | 可复用的 UI 组件 | 新增或修改组件时 |
| `test/` | 自动化测试 | 新增功能或修复缺陷时 |
| `scripts/` | 开发和打包辅助脚本 | 构建流程调整时 |

## 命名约定

- **文件名**：代码文件使用 kebab-case（如 `clipboard.js`），`docs/` 文档文件使用中文名称
- **JavaScript 变量/函数**：camelCase（如 `getAllClips`、`togglePin`）
- **CSS 类名**：kebab-case（如 `.card-item`、`.search-bar`）
- **数据库字段**：snake_case（如 `created_at`、`image_path`）
- **SQL 表名**：复数 snake_case（如 `clips`、`settings`）
- **常量**：UPPER_SNAKE_CASE（如 `MAX_RETENTION_DAYS`）
