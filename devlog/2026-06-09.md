- Electron 的 `globalShortcut.register('Super+Z')` 在 Windows 上不可用（RegisterHotKey 不支持 MOD_WIN），改用 `Ctrl+Shift+Z`
- `--silent` 参数被 Electron/Chromium 自身消耗，改用 `--autostart` 自定义标志
- npm start 中文乱码：Windows cmd GBK 编码，添加 `chcp 65001`
- 托盘图标无文件时显示空白，生成 16x16 / 32x32 PNG 后解决
- 固定复选框初始不可点击：`-webkit-app-region: drag` 覆盖了整个 `top-fixed`，需给复选框加 `no-drag`
