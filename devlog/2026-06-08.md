## 遇到的问题
1. better-sqlite3 编译失败 → 改用 sql.js（纯 JS）
2. npm 下载 Electron 超时 → 切换镜像源
3. `require('electron')` 返回字符串 — `ELECTRON_RUN_AS_NODE=1` → spawn 子进程绕过
4. sql.js `last_insert_rowid()` 在 `db.export()` 后重置 → 先获取ID再save()
5. jimp 导入方式：需 `const { Jimp } = require('jimp')`
6. 托盘图标无文件 → 用 jimp 运行时生成图标
7. Win+Z 无法注册 → Windows RegisterHotKey 不支持 MOD_WIN，改用 Ctrl+Shift+Z
8. `--silent` 参数被 Electron/Chromium 自身消耗 → 改用 `--autostart`
9. npm start 中文乱码 → Windows cmd GBK 编码，添加 `chcp 65001`
